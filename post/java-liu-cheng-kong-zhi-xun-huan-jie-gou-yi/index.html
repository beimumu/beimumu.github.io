if语句

  1. if单选择结构

    package top.beimumu.struct;

import java.util.Scanner;

public class IfDemo {
    public static void main(String[] args) {
        //if单选择结构：从上而下执行。
        Scanner scanner = new Scanner(System.in);
        System.out.println("请输入内容：");
        String s = scanner.nextLine();
        //equals:判断字符串是否相等。判断字符串不要用=号。
        if (s.equals("Hello")){
            System.out.println(s);
        }
        System.out.println("End");
        scanner.close();
        }
}

  2. if双选择结构：判断学生成绩成绩60分以上合格，60分以下不合格

    package top.beimumu.struct;

import java.util.Scanner;

public class IfDemo02 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("请输入成绩：");
        double score = scanner.nextDouble();
        if (score>=60){
            System.out.println("成绩合格");
        }else{
            System.out.println("成绩不合格");
        }
        scanner.close();
    }
}

  3. if多选择结构

    练习：将成绩划分A–D五个等级

    package top.beimumu.struct;

import java.util.Scanner;

public class IfDemo03 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("请输入成绩：");
        int score = scanner.nextInt();
        if (score==100){
            System.out.println("恭喜满分");
        }else if ((score<100)&&(score>=90)){
            System.out.println("A级");
        }else if ((score<90)&&(score>=80)){
            System.out.println("B级");
        }else if ((score<80)&&(score>=70)){
            System.out.println("C级");
        }else if ((score<70)&&(score>=60)){
            System.out.println("D级");
        }//为了程序的严谨需要给定成绩的合法范围
        else if ((score<60)&&(score>0)){
            System.out.println("A级");
        }else{
            System.out.println("成绩不合法");
        }
        scanner.close();
    }
}

    ==tips:一旦其中一个else if语句检测为true，其他的else if以及else语句都将跳过执行。==

switch多选择结构

  • switch case语句判断一个变量与一系列值中某个值是否相等，每个值称为一个分支。

  • switch语句中的变量类型可以是：

    1. byte，short，int或char；

    2. 从JavaSE7开始

    3. switch支持字符串String类型

      package top.beimumu.Switch;

public class SwitchDemo02 {
    public static void main(String[] args) {
        String name = "琲木木";
        switch (name){
            case "大佬":
                System.out.println("大佬");
                break;
            case "林木木":
                System.out.println("林木木");
                break;
            case "琲木木":
                System.out.println("琲木木");
                break;
            default:
                System.out.println("啥也不是!");
                break;
        }
    }
}

    4. 同时case标签必须为字符串常量或字面量。

  • switch语句语法格式如下：

    switch(expression){
    case value :
        //语句
        break;//可选
    case value :
        //语句
        break;//可选
    //你可以有任意数量的case语句
    default ：//可选
        //语句
}

  • 练习:成绩等级

    package top.beimumu.Switch;

public class SwitchDemo01 {
    public static void main(String[] args) {
        char grade = 'C';
        switch (grade){
            //case穿透：在语句中若无breeak，程序则会一直运行下去并执行对应的case语句及以后的case语句。
            //switch：匹配一个具体的值。
            case 'A':
                System.out.println("优秀");
                break;
            case 'B':
                System.out.println("合格");
                break;
            case 'C':
                System.out.println("良好");
                break;
            case 'D':
                System.out.println("太棒了");
                break;
            case 'E':
                System.out.println("太差");
                break;
            default:
                System.out.println("未知等级");
                break;
        }
    }
}

  • 用idea查看反编译（==看源码的思想==）

    由于JavaSE7后，switch支持字符串类型。但是字符串的本质还是数字，我们要查看具体运行过程则需要进行反编译。

    1. 在项目结构中找到project compiler output的文件路径。

    2. 找到文件后打开后对应.class文件会发现是乱码的。

    3. 将文件拖到idea上就会看到反编译后的文件。

    4. 每一个字符都有对应的hashcode

while和do…while语句

  • while先判断后执行

    package top.beimumu.struct;

public class WhileDemo01 {
    public static void main(String[] args) {
        int i = 0;
        int sum = 0;

        while (i<=100){
            sum+=i;
            i++;
        }
        System.out.println(sum);
    }
}

  • do..while先执行后判断

    package top.beimumu.struct;

public class DoWhileDemo01 {
    public static void main(String[] args) {
        int i = 0;
        int sum = 0;
        do {
            sum+=i;
            i++;
        }while (i<=100);
        System.out.println(sum);
    }
}

for循环

  • for循环语句是支持迭代的一种通用结构，是最有效、最灵活的循环结构。

  • for循环执行的次数是在执行前就确定的，语法格式如下：

    for(初始化;布尔表达式;更新){
    //代码语句
}

  • 快捷键100.for就会生成

    for (int i = 0; i < 100; i++) {
            
        }

  • 练习1：计算1-100的奇数和偶数的和

    package top.beimumu.struct;

public class ForDemo01 {
    public static void main(String[] args) {
        //定义一个奇数和
        int oddsum = 0;
        //定义一个偶数和
        int evensum = 0;
        //for语句
        for (int i = 0; i <=100; i++) {
            if (i%2!=0){//不能被2除尽，表示奇数
                oddsum+=i;
            }else{
                evensum+=i;
            }
        }
        System.out.println("奇数和="+oddsum);//奇数和=2500
        System.out.println("偶数和="+evensum);//偶数和=2550
    }
}

  • 练习2：用while或for循环输出1-1000之间能被5整除的数，并且每行输出3个

    package top.beimumu.struct;

public class ForDmo02 {
    public static void main(String[] args) {
        for (int i = 0; i <= 1000; i++) {
            if (i%5==0){
                System.out.print(i+"\t");//不会换行，数值会横排下去
            }
            if (i%(5*3)==0){//为了每输出三个（5*3=15）换行
                System.out.println();//就相当于i能被15整除后就会换行
                //System.out.print("\n");
            }
        }
    }
}

foreach语句（增强for循环）

  • 主要用在数组或集合上

  • foreach语法格式：

    for(声明语句:表达式){
    //类型 变量名：表达式
    //代码句子
}

  • 例子

    package top.beimumu.struct;

public class ForDemo04 {
    public static void main(String[] args) {
        //方法一
        int[] numbers = {10,20,30,40,50};
        for (int i=0;i<5;i++){
            System.out.println(numbers[i]);
        }
        System.out.println("========");
        //方法二
        //遍历数组的元素。将numbers赋给x并且运行numbers中的所有数组。
        for (int x:numbers){
            System.out.println(x);
        }
    }
}

break、continue、goto关键字

  • 一个辞职，一个请假：

    1. break：在所有循环语句中可以起到==终止此次循环体的运行==（循环体外的后面的继续运行）

    2. continue：在只能在循环语句中起==跳过此次的运行==（循环体外的后面的不运行）且放回最初的地方继续运行。

    3. break:终止当前循环，for循环后面的程序将不再执行。

      public class Demo16 {
    public static void main(String[] args) {
        for (int i = 1; i < 10; i++) {
            if (i==4){
                System.out.println("终止后面的程序，不执行！");
                break;
            }
            System.out.println(i);
        }
    }
}

    4. continue：跳过后面的执行语句。

      public class Demo16 {
    public static void main(String[] args) {
        for (int i = 1; i < 10; i++) {
            if (i==4){
                System.out.println("跳过后面的程序，不执行！");
                continue;
            }
            System.out.println(i);
        }
    }

    1. goto关键字通俗的来说就是带标签的continue，因为很复杂所以不研究。
    2. 对Java来说唯一用到标签的地方就是在循环语句之前。

      双层嵌套for循环

  1. 格式：

    for(;;){
    for(;;){
      //循环体;  
    }
}
    • 外层for循环控制循环的行数，内层for循环控制循环每行的内容。
    • 外层for循环一次，内层for循环一周。
    public class Demo04 {
    public static void main(String[] args) {
        for (int i = 1; i < 10; i++) {
            for (int j = 1; j <= i; j++) {
                //内层for循环控制循环每行的内容,所以没有ln
                System.out.print(j+" ");
            }
            //外层for循环控制循环的行数，我不需要他输出内容，所以没有内容但有ln
            System.out.println();
        }
    }
}

  3. 练习一：打印三角形

    package top.beimumu.struct;
/**
* 000001
* 0000111
* 00011111
* 001111111
* 0111111111
*/
public class TestDemo {
    public static void main(String[] args) {
        //打印三角形 5行
        for (int i = 1; i <= 5; i++) {
            for (int j = 5; j >= i; j--) {
                System.out.print("0");
            }
            for (int j = 1; j <=i ; j++) {
                System.out.print("1");
            }
            for (int j = 1; j < i; j++) {
                System.out.print("1");
            }
            //1.相当于弄了5*5的正方形
            System.out.println();
        }
    }
}
