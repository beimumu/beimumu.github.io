2.3 List接口

2.3.1 ArrayList容器类

  • ArrayList是一个动态数组。支持可随需求增长的动态数组。

  • 语法结构

    ArrayList<泛型> list = new ArrayList<>();//jdk1.7后，new后的泛型可以省略。

  • 添加

    list.add();

    list.add("张三");
list.add("王五");
list.add("赵六");
System.out.println(list); //[张三, 王五, 赵六]

  • 插入

    list.add(index,element); 不会覆盖之前的内容，后面的内容向后移一位。

    list.add(1,"李四");
System.out.println(list); //[张三, 李四, 王五, 赵六]

  • 删除

    list.remove();

    1、根据索引删除指定元素，返回被删除的元素。

    String remove1 = list.remove(3);
System.out.println(remove1); //赵六

    2、删除元素,返回true或false。

    boolean remove2 = list.remove("王五");
System.out.println(remove2); //true

    3、逆向遍历删除指定元素

 *
 * 4   3

 * 2   1
 * 1   0
 * 0

    if (list.get(re).equals("王五")){
        boolean remove = list.remove("王五");
        System.out.println(remove);
    }
}

2.3.2 Vector容器类

  • Vector底层也是数组实现的。

  • 相关方法都加了同步检查，关键字synchronized(英[ˈsɪŋkrənaɪzd])同步标记。

    • 线程更安全，但是效率也变低了（并行改串行）。

  • 同ArrayList用法一致都是实现了List接口。

  • 只要不是多个线程同时更改数据，我们一般还是用ArrayList容器类。

    注释：在大部分的需求中，我们还是常用ArrayList，对于ArrayList，我们主要用来存储数据，并且读取数据，所以考虑性能，而不考虑线程安全性的问题。但如果多个线程需要对数据修改，此时就需要考虑线程安全了。

  • Vector与ArrayList底层源码类似，但也有不同点。

    • 初始化方面：Vector是立即初始化，ArrayList是延迟初始化，即啥时候用啥时候扩容。
    • 扩容操作：Vector是进行三目运算符，ArrayList是进行右移（除以2）操作。
    • 扩容倍数：Vector是2倍扩容，ArrayList是1.5倍扩容。
    • 计数方面：Vector中elementCount和ArrayList中size作用一致。

2.3.3 Stack容器

  • Stack容器（栈容器）是Vector容器类的子类。

  • 实现了标准的先进后出的栈。

  • List接口的方法通用，除此之外，通过5个操作方法对Vector进行扩展，所以允许向量视为堆栈。

    Stack<String> stack = new Stack<>();

//List接口的方法
stack.add("a");
stack.add("b");
stack.add("c");
stack.add("d");

String get = stack.get(0);
System.out.println(get);

//Stack容器的方法
//1、压栈，将元素存到栈顶。
stack.push("a");
stack.push("b");
stack.push("c");
stack.push("d");

//2、弹栈，将元素从栈顶删除，并返回删除元素。
String pop = stack.pop();
System.out.println("删除栈顶元素："+pop);

//3、查看栈顶元素，但不删除。
String peek = stack.peek();
System.out.println("查看栈顶元素："+peek);

//4、根据元素找到在栈中的位置，栈从1开始
int search = stack.search("a");
System.out.println(search);

//5、判断栈是否为空
boolean empty = stack.empty();
System.out.println(empty);

System.out.println(stack);
2.3.1 栈的实例：“字符串匹配算法1.0.0”
//字符串匹配问题,匹配是指不同类型的括号必须左右呼应，可以相互包含，但不能交叉。
//例如：
//        ..(..[..]..)..  是允许的
//        ..(...[...)....].... 是禁止的
public class StackTesst {
    public static void main(String[] args) {
        StackTesst st = new StackTesst();
        st.symetry();
    }

    public void symetry(){
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入符号：");
        String str = sc.nextLine();
        
        Stack<String> stack = new Stack<>();

        //假设修正法
        boolean flag = true;//假设匹配为真
        
        for (int i=0; i < str.length(); i++){
            int c = str.charAt(i);
            if (c == '{'){
                stack.push("}");
            }
            if (c == '['){
                stack.push("]");
            }
            if (c == '('){
                stack.push(")");
            }
            //判断匹配问题
            if (c == '}' || c == ']' || c == ')'){
            /*----------校验操作：这是遍历前，判断栈内是否为空------------*/  
                //栈为空，字符串右括号在栈里没有对应的左括号，此时有两种情况。
                //1、右括号开头
                //2、右括号多余。
                if (stack.empty()){
                    flag = false;
                    break;
                }
                String x = stack.pop();
                if (x.charAt(0) != c){ // 栈不为空，弹出的元素与c作比较，相等就说明字符串中括号匹配，反之不匹配。
                    flag = false;
                    break;
                }
            }
        }
        
        /*----------校验操作：这是遍历完成后，再次判断栈内是否为空-------*/  
        //为空取反，不为空就说明栈里还有元素（右括号），与字符串元素（左括号）不匹配，此时有两种情况。
        // 1、字符串左括号多余。
        //2、字符串就直接没有匹配的元素
        if (!stack.empty()){
            flag = false;
        }
      System.out.println(flag);
        sc.close();
    }
}

public class StackTest {
    public static void main(String[] args) {

        StackTest st = new StackTest();

    public int symetry(){

        Scanner sc = new Scanner(System.in);
        System.out.println("请输入需要判断的符号：");
        String str = sc.nextLine();
        /*String str = "{}[](";*/
        Stack<String> stack = new Stack<>();

        //判断标志位
        int flag = 0 ;
        //接收字符
        char getStr = ' ';

        //将字符串传入栈中
        for (int i=0; i < str.length();i++){
            int c = str.charAt(i);
            if (c == '{'){
                stack.push("}");
            }
            if (c == '['){
                stack.push("]");
            }
            if (c == '('){
                stack.push(")");
            }
            if (c == '}' || c == ']' || c == ')'){
                if (stack.empty()){
                    //问题一：将字符转换为字符串，根据索引找字符????为啥转换不了
                    /*getStr = str.valueOf(c);*/
                    getStr = str.charAt(i);

                    flag = 1;//栈内为空，右括号多余，不匹配  "{[---右括号多余---]})"
                    break;

                }
                String x = stack.pop();

                if (x.charAt(0)!= c){

                    getStr = x.charAt(0);
                    flag = 2; // 弹出的内容与字符串不同，不匹配 "{---与弹出的括号不匹配---)"
                    break;

                }
            }
        }
        if (!stack.empty()){
            getStr = str.charAt(0);
            flag = 3; //栈内不为空，则左括号多余，不匹配 "{[---左括号多余---]}("
        }

        if (flag == 0){
            System.out.println("字符串匹配成功"); //"{[(hello,java)]}"
            return flag;
        }else if (flag == 1){
            System.out.println("提示"+flag+"：栈为空，括号“ "+getStr+" ”多余"); 
            System.out.println("栈内元素为："+stack);
            return flag;
        }else if (flag == 2){
            System.out.println("提示"+flag+"：栈不为空，弹出括号“ "+getStr+" ”没有匹配");
            System.out.println("栈内元素为："+stack);
        }else if (flag == 3){
            System.out.println("提示"+flag+"：栈内不为空，括号“ "+getStr+" ”多余");
            System.out.println("栈内元素为："+stack.peek());
            return flag;
        }
        //System.out.println("输入有误"+flag);
        sc.close();
        return flag;
    }
}

2.3.4 LinkedList容器类

  • LinkedList是不连续的双向链表。

  • ArrayList与LinkedList的区别

    区别 ArrayList LinkedList
    底层不同 数组 双向链表
    查询效率 高（遍历索引） 低（折半查询Node结点）
    插入删除效率 低（会前移、后移） 高（断链、指向）
    线程安全 不安全 不安全

    总结：

    1、数据不变，用于查询选择ArrayList。

    2、数据经常需要修改，选择LinkedList。

  • LinkedList实现了List接口，所以也就可以调用List的方法，同时LinkedList也有自己特有的方法，为了方便访问和操作。

    private static class Node<E> {
    E item; //数据域
    Node<E> next; // 指针，指向下一结点
    Node<E> prev; // 指针，指向上一结点

    //带参的构造方法
    Node(Node<E> prev, E element, Node<E> next) {
        this.item = element;
        this.next = next;
        this.prev = prev;
    }
}

    6、插入指定序列add(int index, E element)

      /**
   * Inserts the specified element at the specified position in this list.
   * Shifts the element currently at that position (if any) and any
   * subsequent elements to the right (adds one to their indices).
   *
   * @param index index at which the specified element is to be inserted
   * @param element element to be inserted
   * @throws IndexOutOfBoundsException {@inheritDoc}
   */
  public void add(int index, E element) {
      //校验索引位置是否合法
      checkPositionIndex(index);
  
      //索引等于元素个数，比如4,4;在尾结点加入新元素。
      if (index == size)
          linkLast(element);
      else
      //不等于时，进行折半查询
          linkBefore(element, node(index));
  }
  ----------------------------------------------------------------------
  private void checkPositionIndex(int index) {
      //取反操作
      if (!isPositionIndex(index))
          throw new IndexOutOfBoundsException(outOfBoundsMsg(index));
  }
  
  /**
   * Tells if the argument is the index of a valid position for an
   * iterator or an add operation.
   */
  private boolean isPositionIndex(int index) {
      //这里可以等于size，是因为判断的是元素个数，而不是索引。
      return index >= 0 && index <= size;
  }
  
  private String outOfBoundsMsg(int index) {
      return "Index: "+index+", Size: "+size;
  }

  //node折半查询
  /**
   * Returns the (non-null) Node at the specified element index.
   */
  Node<E> node(int index) {
      // assert isElementIndex(index);
      //界定条件：(size >> 1)
      if (index < (size >> 1)) {
          //满足条件，从头结点开始找。
          Node<E> x = first;
          for (int i = 0; i < index; i++)
              //类似于指针后移的操作
              x = x.next;
          return x;
      } else {
          //不满足条件，从尾结点开始找。
          Node<E> x = last;
          for (int i = size - 1; i > index; i--)
              //类似于指针前移的操作
            x = x.prev;
          return x;
    }
  }
/**
   * Inserts element e before non-null Node succ.
   */
  void linkBefore(E e, Node<E> succ) {
      // assert succ != null;
      final Node<E> pred = succ.prev;//succ的前驱赋值给pred(即两个寻找挂接的过程)
      final Node<E> newNode = new Node<>(pred, e, succ);//成功后，新结点进行连接结点pred和结点succ的地址（先出，左出）
      succ.prev = newNode;//断链，succ前驱指向新结点（后进，右进）
      if (pred == null)
          first = newNode;//此时succ前面没有结点，新结点成为头结点。
      else
          pred.next = newNode;//断链，pred的后继指向新结点。
      size++;
      modCount++;
  }

 * @param index the index of the element to be removed
 * @return the element previously at the specified position
 * @throws IndexOutOfBoundsException {@inheritDoc}
 */
public E remove(int index) {
    //校验元素位置是否合法
    checkElementIndex(index);
    return unlink(node(index));
}
---------------------------------------------------------------------------
private void checkElementIndex(int index) {
    if (!isElementIndex(index))
      throw new IndexOutOfBoundsException(outOfBoundsMsg(index));
}
---------------------------------------------------------------------------
/**
 * Tells if the argument is the index of an existing element.
 */
private boolean isElementIndex(int index) {
    //这里不能等于size，是因为判断的是索引，而不是元素个数。与前面的checkPositionIndex()方法区分开
    return index >= 0 && index < size;
}
---------------------------------------------------------------------------
/**
 * Returns the (non-null) Node at the specified element index.
 */
Node<E> node(int index) {
    // assert isElementIndex(index);

    if (index < (size >> 1)) {
        Node<E> x = first;
        for (int i = 0; i < index; i++)
            x = x.next;
        return x;
    } else {
        Node<E> x = last;
        for (int i = size - 1; i > index; i--)
            x = x.prev;
        return x;
    }
}
---------------------------------------------------------------------------
/**
 * Unlinks non-null node x.
 */
E unlink(Node<E> x) {
    // assert x != null;
    final E element = x.item;
    final Node<E> next = x.next;
    final Node<E> prev = x.prev;

    if (prev == null) {
        first = next;
    } else {
        prev.next = next;
        x.prev = null;
    }

    if (next == null) {
        last = prev;
    } else {
        next.prev = prev;
        x.next = null;
    }

    x.item = null;
    size--;
    modCount++;
    return element;
}

 * Retrieves and removes the head (first element) of this list.

 * @return the head of this list
 * @throws NoSuchElementException if this list is empty
 * @since 1.5

public E remove() {
    return removeFirst();

/**
 * Removes and returns the first element from this list.
 *
 * @return the first element from this list
 * @throws NoSuchElementException if this list is empty
 */
public E removeFirst() {
    final Node<E> f = first;
    if (f == null)
        throw new NoSuchElementException();
    return unlinkFirst(f);
}
/**
 * Unlinks non-null first node f.
 */
private E unlinkFirst(Node<E> f) {
    // assert f == first && f != null;
    final E element = f.item;
    final Node<E> next = f.next;
    f.item = null;
    f.next = null; // help GC
    first = next;
    if (next == null)
        last = null;
    else
        next.prev = null;
    size--;
    modCount++;
    return element;
}

2.3.5 List容器类练习题

2.3.5.1 练习一：

创建一个ArrayList来存储Person对象（姓名、年龄、id）。

排序规则：年龄升序，年龄相等时，姓名按字典顺序排序。

用Collections工具类正序排列，并打印结果。

package top.beimumu.CollectionDemo_03;

public class Person implements Comparable<Person> {

    private String name;
    private int age;
    private int id;

    @Override
    //年龄升序，年龄相等时，名字按字典顺序。
    public int compareTo(Person o) {

        if (this.age > o.getAge()){
            return 1;
        }
        if (this.age == o.getAge()){
            return this.name.compareTo(o.getName());
        }
        return -1;
    }

    public Person() {
    }

    public Person(String name, int age, int id) {
        this.name = name;
        this.age = age;
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (o == null || getClass() != o.getClass()) return false;

        Person person = (Person) o;

        if (age != person.age) return false;
        if (id != person.id) return false;
        return name != null ? name.equals(person.name) : person.name == null;
    }

    @Override
    public int hashCode() {
        int result = name != null ? name.hashCode() : 0;
        result = 31 * result + age;
        result = 31 * result + id;
        return result;
    }

    @Override
    public String toString() {
        return "Person{" +
                "name='" + name + '\'' +
                ", age=" + age +
                ", id=" + id +
                '}';
    }
}
