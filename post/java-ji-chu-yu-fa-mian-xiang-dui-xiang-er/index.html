6.字符串

1.String的创建方式

  • 直接赋值：String a = "hello";

    先去字符串常量池查找有没有hello这个字符串，如果有，那么就让变量a指向这个字符串；如果没有，就创建一个hello存入常量池中。

  • 创建后赋值：String b = new String("hello") ;

    在堆中开辟一个空间存放hello这个字符串，然后回去常量池中检查有没有hello这个字符串，如果有什么也不用做；如果没有，就创建一个hello存入常量池。

  • 在Java中，只要是new出来的东西，那么一定会出现在堆内存中，并且new几个，就会在堆内存中出现几个。

String a = "hello";
String b = new String("hello");
System.out.println(a.equals(b));//true
System.out.println(a==b);//false

==与equals()的区别：

  1. 对于==：
    • 如果是两个基本数据类型进行比较，那么比较的是值。
    • 如果是两个引用数据类型进行比较，那么比较的是（引用地址）内存地址。
  2. 对于equals()：
    • 在Object类(没有重写equals())，比较的是两个对象的引用地址。
    • 在String类（重写equals()），比较的是两个对象的字符串的内容（字面量）是否一样。
  3. 在Object类中，因为equals()方法没有重写,所以和==都是比较两个对象的引用地址。
public class Demo01 {
    public static void main(String[] args) {
        String a = "abc";
        String b = "abc";
        String c = new String("abc");
        String d = new String("abc");

        System.out.println(a.equals(b)); //true
        System.out.println(b.equals(c)); //true
        System.out.println(b.equals(d)); //true
        System.out.println(c.equals(d)); //true

        System.out.println(a == b); //true
        System.out.println(b == c); //false
        System.out.println(b == d); //false
        System.out.println(c == d); //false
    }
}

3.String对象的常用方法

length()：返回字符串的长度（个数）

String str = "hello world 你好 世界 !";
int i = str.length();
System.out.println(i);//结果返回12

charAt(index)：返回指定的下标索引index处的字符char（根据索引查找字符）。

//声明char型，返回下标为2的字符。
        char c1 = str.charAt(2);
        System.out.println(c1);

//声明int型，返回下标为2的字符的ASCII码。
        int c2 = str.charAt(2);
        System.out.println(c2);

indexOf(int i/[char c]/[String str])：返回指定字符在字符串中第一次出现的下标索引（根据字符查找索引）。

int index1 = str.indexOf(108);//参数给int ASCII码
System.out.println(index1);

int index2 = str.indexOf('l');//参数给char
System.out.println(index2);

int index3 = str.indexOf("l");//参数给String
System.out.println(index3);

indexOf(int i/[char c]/[String str],int fromIndex)：从指定下标处，开始查找指定字符第一次出现的索引。

int index4  = str.indexOf(108, 5);//参数给int ASCII码
System.out.println(index4);

int index5  = str.indexOf('l', 5);//参数给char
System.out.println(index5);

int index6 = str.indexOf("l", 5);//参数给String
System.out.println(index6);

lastIndexOf(char c)：返回指定的字符在字符串中最后一次出现的索引。（同indexOf();用法相同）

int index7 = str.lastIndexOf('l');
System.out.println(index7);//结果为9

concat(str)：将指定的字符串拼接到字符串的结尾。（同连接符“+”用法相同）

String concat = str.concat("\t hello java!");
System.out.println(concat);//"hello world 你好 世界 !	hello java!"

equals(str)：判断两个字符串的字面量是否相等

String str1 = "Hello Whorld";
System.out.println(str.equals(str1));//false

toUpperCase(str); 将字符串转为大写

String upperCase = str.toUpperCase();
System.out.println(upperCase);//"HELLO WORLD 你好 世界 !"

toLowerCase(str); 将字符串转为小写

String lowerCase = upperCase.toLowerCase();
System.out.println(lowerCase);//"HELLO WORLD 你好 世界 !" ->"hello world 你好 世界 !"

contians(CharSequence s); 字符串中是否包含指定的**字符序列(CharSequence),**返回Boolean值。

boolean contains = str.contains("llo w");
System.out.println(contains); //true

substring(index); 从index处(包含index处)开始截取字符串，一直截取到最后。

String substring = str.substring(1);
System.out.println(substring);//"ello world!"

substring(int beginIndex, int endIndex); 从beginIndex处(包含beginIndex处)开始截取字符串，一直截取到endIndex(不包endIndex)之前。

String substring1 = str.substring(1, 7);
System.out.println(substring1); //"hello world!" -> "ello w"

split(str); 根据给定的正则表达式拆分字符串。结果是一个字符串的数组。

String[] split = str.split(" ");//以空格为拆分符。

//方法一、用数组方法打印数组
System.out.println(Arrays.toString(split)); //[hello, world, 你好, 世界, !]

//方法二、增强for循环遍历数组
for(String str:split){
    System.out.println(str);
}

compareTo(String anotherString); 按照字典顺序比较两个字符串，用比较的字符串减去被比较的字符串的差值。

int compareTo = str.compareTo("aello world!");
System.out.println(compareTo); // a-h => 7

trim(); 去除掉字符串两端的空格。

String s = " hehe haha ";
String trim = s.trim();
System.out.println(trim+"   "+trim.length());//"hehe haha" 字符串长度：11->9

starsWith(str); 检测字符串是否以str开头。

boolean startsWith = s.startsWith(" hehe");
System.out.println(startsWith);//true

endswith(str); 检测字符串是否以str结尾。

boolean endsWith = s.endsWith("haha ");
System.out.println(endsWith);//true

replace(被替换的内容，替换的新内容); 替换

String replace = s.replace(" ", "/");
System.out.println(replace);// "/hehe/haha/"
String replace1 = s.replace("haha", "哈哈");
System.out.println(replace1); //" hehe 哈哈 "

replaceFirst(); 只替换所选择的第一个的内容

String replaceFirst = s.replaceFirst(" ", "/");
System.out.println(replaceFirst); //"/hehe haha "

replaceAll(); 替换掉所选择的内容

String replaceAll = s.replaceAll(" ", "abc");
System.out.println(replaceAll); //"abcheheabchahaabc"

7.封装

  1. OOP(Object Oriented Programming)：面向对象编程
  2. 面向对象编程的三大特性：封装、继承、多态。

1.封装的概念

  • 将东西打包装在一起，以新的形式展现出来
  • 将重要的信息隐藏，不让外界看到
  • 将数据和方法包装在类中，再加以隐藏。

2.属性私有化

  • private关键字修饰的属性，只能在本类中被访问。

    public class Student {
    //属性私有化：private修饰的属性，只能在本类中被访问。
    private String name;

    //如果想让外界访问到，则通过方法来实现。
    //1、给属性赋值
    public void setName(String name) {
        this.name = name;
    }
    
    //2、获取属性值
    public String getName() {
        return name;
    }
}
---------------------------------------
public class TestStudent {
    public static void main(String[] args) {
        Student stu = new Student();
        stu.setName("张三");
        System.out.println(stu.getName());
    }
}

    需要注意的是：

    1. 给属性赋值(Setter)方法中的返回值是void，而获取属性值(Getter)方法中就需要根据返回的数据类型来判断。
    2. 属性私有化后name属性就不能通过stu.name();来调取了，只能通过stu,setName();来访问；同理，打印语句System.out.println(stu.name());也变成了System.out.println(stu.getName());

3.构造方法(构造器 constructor)

  1. 语法规则

    访问修饰符 方法名（与类名相同）{
    
}

  2. 特点

    • 构造方法的方法名与类名完全相同
    • 构造方法没有返回值类型

  3. 构造方法的调用:

    使用new 操作符。比如Animal a = new Animal();

  4. 构造方法可以重载，可以有：无参数的构造方法，也可以有带参数的构造方法。

  5. 构造方法的主要作用：给属性赋合适的初值，而不是创建对象。

    装修毛坯房：装修工（构造器）进行装修房子（对象），房子你已经买了，装修工只负责装修，不负责买房。

  6. 举例

    public Animal(String name,int legCount,String color) {
    this.name = name;
    this.legCount = legCount;
    this.color = color;
}
//初始化对象，即创建对象后，调用有参构造方法给b进行赋初值。
Animal b = new Animal("奶酪", 4, "黄色");

    分析 过程：

    1.在栈内存中开辟空间，保存Animal b 这个引用变量。

    2.在堆中给属性开辟空间。

    3.调用构造方法。如果调用的是带参的构造方法，把传进来的参数，挨个的赋值到堆中的属性上。

    4.让栈中的引用变量，指向堆中的对象。

    其中，1、2创建对象实例化的过程，3、4初始化的过程。

  7. 每一个类，如果你没有声明任何的构造方法，那么JVM会默认帮你生成一个公有的无参的构造方法。并且创建对象的时候，会自动去调用它。如果你在类中声明了构造方法，那么JVM就不会再帮你生成任何构造方法了。

  8. 在类的构造方法中，可以使用this()来调用本类中的其他的构造方法，这条语句要求是构造方法中的第一条语句。

  9. 构造方法与普通方法的区别：

    1. 语法上。无返回值类型。方法名与类名完全一致。
    2. 调用上。普通方法可以用对象来调用。构造方法只能在创建对象的对象使用new操作符来调用。
    3. 每次构造方法被执行，都会有一个对象被创建。普通方法可以使用对象进行无限次的调用。

4.toString()方法

  • 练习创建一个电脑类,要求有品牌，型号，价格,颜色等属性。实现对属性的封装。
    要求，创建对象的时候，可以快速的给这些属性赋值。

    //Computer类
public class Computer {
    private String brand; //品牌
    private String type; //型号
    private double price; //价格
    private String color; //颜色

    public Computer() {
        System.out.println("这是无参的构造函数");
    }

    public Computer(String brand, String type, double price, String color) {
        System.out.println("这是有参的构造函数");
        this.brand = brand;
        this.type = type;
        this.price = price;
        this.color = color;
    }

    public String getBrand() {
        return brand;
    }

    public void setBrand(String brand) {
        this.brand = brand;
    }

    public String getType() {
        return type;
    }

    public void setType(String type) {
        this.type = type;
    }

    public double getPrice() {
        return price;
    }

    public void setPrice(double price) {
        this.price = price;
    }

    public String getColor() {
        return color;
    }

    public void setColor(String color) {
        this.color = color;
    }

    @Override
    public String toString() {
        return  "品牌：" + brand +
                " 型号：" + type +
                " 价格：" + price +
                "元 颜色：" + color
                ;
    }
}
    //TestComputer类
public class TestComputer {
    public static void main(String[] args) {
        //第一种：使用new创建对象，没有初始化，所以调用了无参的构造函数。
        //没有通过toString()方法，所以不能直接通过调用对象来打印。
        Computer c1 = new Computer();
        c1.setBrand("小米");
        c1.setType("小米10");

        //通过toSting()方法来实现对象的调用进行打印，此过程中。
        Computer c2 = new Computer("华为","华为p50",5000,"黑色");

}
